To find a dimension:

New Height = Width ¬∑ (Aspect Ratio Height / Aspect Ratio Width)

New Width = Height ¬∑ (Aspect Ratio Width / Aspect Ratio Height)

To find the Aspect Ratio:

GCD = Greatest Common Divisor(Width, Height)

Aspect Ratio = (Width / GCD) : (Height / GCD)

Reference: Wikipedia - Aspect Ratio
